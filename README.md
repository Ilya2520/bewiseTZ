# bewiseTZ
Тестовое задание

## Запуск

'docker-compose up --build'

## Docker-compose
Содержит два контейнера. Приложение и бд postgres. Контейнер базы данных содержит volume, что позволяет сохранять данные после остановки работы контейнера.

## Директория app
* crud.py
* database.py
* main.py
* requirements.txt
